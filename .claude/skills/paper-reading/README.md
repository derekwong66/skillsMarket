# Paper-Reading 文献阅读技能

一个帮助你系统化阅读和分析学术文献的 Claude Code 技能。

## 功能特点

- 📚 自动提取文献基本信息（题目、作者、来源、摘要等）
- 📋 结构化分析论文的各个部分
- 🎯 识别研究目的、方法和问题
- 💡 提炼理论启示和创新亮点
- 🤔 提供批判性思考和个人研究关联

## 使用方法

### 方式 1：调用技能
```
/paper-reading
```

然后提供文献内容、文件路径或 URL。

### 方式 2：直接提供文献
直接粘贴文献的文本内容，技能会自动分析。

### 方式 3：提供文件
```
/paper-reading /path/to/paper.pdf
```

### 方式 4：提供链接
```
/paper-reading https://arxiv.org/abs/xxxx.xxxxx
```

## 输出内容

技能将生成一份结构化的 Markdown 笔记，包含：

1. **基本信息** - 题目、作者、来源、摘要、关键词
2. **核心内容** - 论文各部分结构概述
3. **研究背景** - 宏观与微观背景
4. **研究目的** - 总体目标与具体目标
5. **研究方法** - 方法选择与详细描述
6. **研究问题** - 核心问题与子问题
7. **研究结论** - 主要结论与详细阐述
8. **理论启示** - 理论贡献与借鉴意义
9. **创新亮点** - 内容、方法、理论创新
10. **自我思考** - 研究评价与个人关联

## 支持的文献格式

- PDF 文件
- Markdown 文件
- 纯文本文件
- Web URL（自动抓取）

## 示例

```
用户: /paper-reading

助手: 请提供你想要阅读的文献（内容、文件路径或 URL）。

用户: 这是一个关于深度学习在自然语言处理中应用的论文...

助手: [生成结构化的文献阅读笔记]
```
