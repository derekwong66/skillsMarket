# Paper-Reading 文献阅读技能

一个帮助你系统化阅读和分析学术文献的 Claude Code 技能。

## 功能特点

- 自动提取文献基本信息（题目、作者、来源、摘要等）
- 结构化分析论文的各个部分
- 识别研究目的、方法和问题
- 提炼理论启示和创新亮点
- 提供批判性思考和个人研究关联

## 使用方法

### 方式 1：调用技能后提供内容
```
/paper-reading
```
然后提供文献内容、文件路径或 URL。

### 方式 2：直接提供文件
```
/paper-reading /path/to/paper.pdf
```

### 方式 3：提供链接
```
/paper-reading https://arxiv.org/abs/xxxx.xxxxx
```

## 支持的文献格式

- PDF 文件
- Markdown 文件
- 纯文本文件
- Web URL（自动抓取）

## 输出内容

技能将生成一份结构化的 Markdown 笔记，包含以下 10 个部分：

1. **基本信息** - 题目、作者、来源、摘要、关键词、重要指数
2. **核心内容** - 论文各部分结构概述（引言、综述、方法、结果、讨论、结论）
3. **研究背景** - 宏观与微观背景分析
4. **研究目的** - 总体目标与具体目标
5. **研究方法** - 方法选择依据与详细描述
6. **研究问题** - 核心问题与子问题
7. **研究结论** - 主要结论概括与详细阐述
8. **理论启示** - 对相关理论的贡献与借鉴意义
9. **创新亮点** - 内容、方法、理论方面的创新
10. **自我思考** - 研究评价与个人研究的关联

## 文件保存

笔记将自动保存到 `doc/` 文件夹，命名格式：`{年份}_{第一作者姓氏}_{简短标题}.md`

示例：`2024_Vaswani_Attention_Is_All_You_Need.md`
